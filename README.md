🔬 Featured Project: The Microscope BuddyGoal: To automate the tedious process of manual biological focal-plane adjustment using Computer Vision and Machine Learning.The ApproachUsing mathematical profiles modeled after the BBBC005 (Broad Bioimage Benchmark Collection), I developed a "Ranger" logic system. This system analyzes the Laplacian Variance (Edge Density) of a sample to determine its focal state.Model: Random Forest Classifier.Logic: * Search State: Rapid Z-axis movement for out-of-focus samples.Coarse Adjust: Controlled micro-stepping as focus improves.Identify: High-precision lock for data collection.Results: Achieved high robotic decision accuracy, validated through Confusion Matrix and K-Means Clustering.Getty Images📑 Lab SummariesLabFocus AreaKey LearningLab 1-10FoundationsData cleaning, Linear/Logistic Regression, and Decision Trees.Lab 11OptimizationGrid Search vs. Randomized Search for hyperparameter tuning.Lab 12FairnessAnalyzing algorithmic bias and disparate impact in socio-economic datasets.🛠️ Tools & TechnologiesLanguages: PythonLibraries: Scikit-Learn, OpenCV, Pandas, NumPy, Matplotlib, SeabornEnvironment: VS Code / Jupyter NotebooksData Sources: BBBC005 (Broad Institute), UCI Machine Learning Repository🛡️ AI Disclosure & EthicsAll work in this repository was developed with an emphasis on Transparent AI.AI Usage: I utilized AI assistants (ChatGPT/Gemini) for code structure, synthetic data modeling, and workflow visualization.Ownership: All logic thresholds, data labels, and model evaluations were personally verified and modified to ensure technical accuracy and ethical alignment.
