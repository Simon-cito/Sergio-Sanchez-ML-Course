# Assignment 1: Professional Environment & Version Control Setup

## The Problem
In Machine Learning and Robotics development, maintaining a "Source of Truth" for code and documentation is critical. The objective of this assignment was to establish a professional version control workflow using GitHub and to initialize a development environment capable of handling high-level AI libraries.

---

## My Approach
* **Version Control:** Established a public repository structure to host all course artifacts, facilitating transparency and collaborative potential.
* **Environment Configuration:** Verified the local development environment using a "Hello, Machine Learning" test script to ensure that the Python kernel and Jupyter interface were communicating correctly.
* **Documentation Standards:** Initialized the project with a README-first mentality, ensuring that all technical code is supported by human-readable context.

---

## Results & Insights
* **Success:** Successfully synchronized a local Jupyter Notebook with the GitHub cloud interface.
* **Insight:** I realized that GitHub is more than just a storage site; it is a portfolio. By naming my repository `Sergio-Sanchez-ML-Course`, I am building a professional asset that showcases my journey as an AI & Robotics student.
